1. Java стабильно занимает первое место в индексе популярности языков программирования tiobe. Вы сможете использовать его для написания авто-тестов и в дальнейшем для разработки под любую платформу. Java — хороший язык для начала изучения программирования.
2. Visual Studio Code — редактор исходного кода, разработанный Microsoft для Windows, Linux и macOS.
3. Самая популярная система контроля версий, которой пользуется большинство разработчиков, в том числе создатели операционных систем Android, Linux. Система контроля версий позволяет работать над кодом проекта совместно, не переживая за потерю данных или конфликт версий.
4. Фреймворк (программная библиотека) для автоматизации действий веб-браузера. Самый популярный инструмент для тестирования веб-приложений, предоставляя авто-тестам доступ к браузеру.
5. JSON — текстовый формат обмена данными, основанный на JavaScript.
6. PostgreSQL — свободная объектно-реляционная система управления базами данных. Существует в реализациях для множества UNIX-подобных платформ, включая AIX, различные BSD-системы, HP-UX, IRIX, Linux, macOS, Solaris/OpenSolaris, Tru64, QNX, а также для Microsoft Windows.
7. Умная и удобная среда разработки для Java, включающая поддержку последних технологий и фреймворков. Предоставляет инструменты для продуктивной работы и подходит для создания коммерческих, мобильных и веб-приложений.
8. Swagger - это программная среда с открытым исходным кодом, опирающаяся на обширную экосистему инструментов, которая помогает разработчикам проектировать, создавать, документировать и использовать веб-сервисы RESTful.
9. GitLab — веб-инструмент жизненного цикла DevOps с открытым исходным кодом, представляющий систему управления репозиториями кода для Git с собственной вики, системой отслеживания ошибок, CI/CD пайплайном и другими функциями.
10. HTTP-клиент для тестирования веб-сайтов. С его помощью можно составлять и редактировать простые или сложные HTTP-запросы.
11. Cистема, которая подходит для отслеживания ошибок и управления проектом в компании любого размера.
12. Apache Maven — фреймворк для автоматизации сборки проектов на основе описания их структуры в файлах на языке POM, являющемся подмножеством XML.

Для кого курс?
Этот курс для абсолютных новичков в QA. Мы откроем Вам дверь в мир информационных технологий. Вы увидите, что можно начать с минимальными усилиями, и для этого не надо знать языки программирования.
Курс — это хорошо, но
что я реально получу
после завершения?
Вы получите знания и опыт, а также резюме, с которым вы сможете найти свою первую работу в ИТ с зарплатой в районе $35-$45 в час! Кроме того, вы сможете увеличить ее в ближайший год-два в 2, а то и в 3! раза. Мы вам поможем и в этом!
Как проходят
занятия?
Мы даем вам видео-теорию для самостоятельного просмотра, а потом закрепляем знания на практических занятиях с инструктором. Мы будем тестировать настоящий проект, поэтому можно рассматривать наш курс не просто как теорию, но как настоящую работу!
Что если нам не
понравится курс?
Первая неделя обучения бесплатна. Тем не менее, вы получите представление как устроен Мир Информационных Технологий и решите, хотите ли вы продолжать или нет.
Должен ли я уволиться с
работы, чтобы учиться?
Нет! Чтобы успешно закончить курс, вам всего лишь необходимо учиться по часу в день после работы. Мы даем только самое важное и нужное, чтобы у вас осталось время на свои дела.
