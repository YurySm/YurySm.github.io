Level 1: QA Engineer Course

Virtual classes for people with no experience in IT. We teach you how to become a solid QA specialist without even knowing how to code. Part-time, 8 weeks course includes everything for your success: theory, practice with real projects, resume, interview prep, mentorship.

Real tools and techniques. Tech companies alike environment. Free Career Services. Average compensation is around $45 per hour according to indeed.com and it is just the beginning!

Level 2: QA Test Automation Engineer

Prerequisites: QA Fundamentals knowledge (QA Engineer course or any other, working experience as QA Analyst) After this course you can become QA Automation Engineer ready to hit the job market.

Course includes: basic knowledge of Java programming languages, algorithms, Selenium WebDriver, Cucumber, JUnit\TestNG, Gradle, automation of APIs and databases! You will learn how to write test automation scripts and run them on CI server (Jenkins) utilizing cloud platforms - Selenium Grid and Browserstack. You will be working in a rel project that you can use in your resume! Resume and Interview preparation is also included!
